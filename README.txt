README tMenuRestaurant 
======================

By Edy Cu Tjong

Purpose
-------
An Android application for user list a presentation of food and beverage offerings, stores and calculate total payment.

Software Develop
----------------
Eclipse Galileo
Android 2.2
Java Development Kit 1.6
Window XP SP3

File Installation
-----------------
bin\tMenuRestaurant.apk

Status Order
------------
Red    : Process served food or offerings
Yellow : Already Served food or offerings to customer 
Green  : Already pay from customer
